## 如何发布自己的 Library Package

只需要一个步骤即可

```
project-root
  |-- build
  |     |-- vfui
  |     |    |-- package.json
  |     |    |
```

将`package.json`的`name`属性修改为你组件库名称即可，例如`my-awesome-ui`
